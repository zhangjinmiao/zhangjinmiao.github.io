模块划分
模块释义进度
fast-admin后台管理模块（controller） 
fast-core核心业务模块（按业务划分） 
fast-framework框架模块，提供基础配置，工具类等 
fast-sso-server单点登录-认证中心模块，使用 xxl-sso 
fast-generator代码生成模块，使用 renren-generator 
fast-quartz定时任务模块 
fast-flyway数据库版本管理工具模块
